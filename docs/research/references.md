## 出典の信頼度に関する注記

| 区分 | 信頼度 | 扱い |
|---|---|---|
| 書籍 *The Bullet Journal Method* | 最高（原典） | 方法論・哲学の根拠として最優先 |
| bulletjournal.com（公式） | 最高（一次） | 記号・手順・公式見解の根拠 |
| Carroll の Medium / TED | 高（本人発言） | 思想・経緯の根拠 |
| Wikipedia | 中（要事実確認） | 歴史・数値の参考（数値は他で裏取り推奨） |
| 解説ブログ各種 | 中〜低 | 補足・実例。公式と矛盾する場合は公式優先 |
| アプリレビュー記事 | 中〜低 | 競合分析の参考。実機確認が望ましい |

### 特に注意が必要な箇所

- **数値（Instagram 投稿数 300 万件、Kickstarter 8 万ドル等）**: Wikipedia 由来。実装判断に使うなら一次ソースで再確認すること。
- **「3 回ルール」の正確な条件**: 書籍と複数の二次情報で言及されるが、厳密な閾値表現は版・記事により揺れがある。
- **AM/PM Reflection の所要時間**: 公式は明示しておらず、「5〜15 分」はコミュニティ慣習。
- **記号 `?`（Explore）と 👁（目アイコン）**: 公式サイトに両表記が見られる。コミュニティでは `?` が主流。
- **既存アプリの機能・価格**: 調査時点のもの。アプリは更新されるため、実装判断時は最新を確認。
